Detailed Description

template<typename TProcessorStates, typename TRegOps, typename TSpecRegOps>
class microemu::internal::RegAccess< TProcessorStates, TRegOps, TSpecRegOps >

Provides access to processor registers.

The RegAccess class allows reading and writing values to processor registers. It provides methods to read the value of a register and write a value to a register. The class also handles edge cases where the register ID is out of range (greater than or equal to 16).

Template Parameters
TProcessorStatesThe type of the processor states used by the RegAccess object.
